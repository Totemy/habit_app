# 📊 Habit Tracker App

A habit tracking application built with Vue 3.  
Focused on state management, user interaction, and clean UI.

---
Demo : [link](https://ishabit-app.netlify.app/)
---

## 🚀 Features

- Create and manage habits
- Track progress
- Dynamic UI updates
- Component-based architecture

---

## 🛠 Tech Stack

- Vue 3 (Composition API)
- TypeScript
- TailwindCSS

---

## 🧠 What I focused on

- State management with composables  
- Clean and reusable components  
- UI/UX simplicity and clarity  

---
